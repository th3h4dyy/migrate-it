# Migrate-it

# Pre-requisites

- Install [Node.js](https://nodejs.org/en/) LTS version

# Getting started

- Clone the repository

```
git clone git@github.com:th3h4dyy/migrate-it.git
```

- Install dependencies

```
cd migrate-it
npm i -g
```

- Run the application from any directory in your system

```
migrate-it
```
